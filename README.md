# Credit_Card_Financial_Dashboard
Power BI Dashboard
Built an end-to-end Credit Card Financial Dashboard to analyze customer behavior, transaction trends, and revenue performance
Designed and created relational tables (cc_detail, cust_detail) in MySQL for structured data storage
Imported and processed large CSV datasets using bulk data loading (LOAD DATA INFILE), resolving file permission and data format issues
Implemented incremental data loading (weekly updates) to simulate real-world data refresh scenarios
Performed data cleaning and transformation using Power Query (handled nulls, corrected data types, standardized columns)
Developed data model in Microsoft Power BI by creating relationships using Client_Num
Created DAX measures such as Total Revenue, Total Transactions, Customer Count, and Interest Earned for KPI tracking
Built interactive dashboards with KPI cards, bar charts, line charts, and slicers to visualize trends and segmentation
Analyzed key business insights including revenue by card category, customer demographics, and utilization patterns
Improved decision-making by identifying high-value customers and transaction trends across time periods
